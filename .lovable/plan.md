

# Email Newsletter Template Builder

## Overview
A professional email newsletter template for Kevin Thompson, CPAs and Advisors that matches their website branding, generates AI content, and exports as email-compatible HTML for ESP services.

---

## Core Features

### 1. Newsletter Preview & Editor Page
- Visual preview of the newsletter that renders exactly as it will appear in email clients
- Live editing capability to customize the generated content before export
- Mobile-responsive preview toggle (desktop/mobile view)

### 2. Branding & Design (Matched to harikthompsoncpas.com)
- Color scheme extracted from the client's website (professional blues/greens, warm tones)
- Client logo placement in header (you'll upload this)
- Typography that's email-safe (Arial, Georgia, or similar web-safe fonts)
- Clean, scannable layout following the Clarify Media template structure

### 3. Newsletter Sections (Pre-populated with AI Content)

**Header Section**
- Client logo
- "Prepared for" tagline with firm name
- Contact information and website link

**Section 1: Headlines (3 bullet points)**
- Trump Accounts update
- RMD on Inherited IRA alert
- Wildfire personal casualty loss bulletin

**Section 2: Compelling Story**
- AI-generated 150-word real-world scenario about inherited IRA RMDs
- Following the engaging story format you referenced (problem → solution → outcome)

**Section 3: Key Insights**
- 2-3 quick takeaways extracted from the story
- Formatted as easy-to-scan bullet points

**Section 4: Action Steps**
- 3 concrete steps readers can take today
- Clear "Step 1, Step 2, Step 3" format
- "Need help? Book your call now" transition

**Section 5: Humor/Levity Section**
- Placeholder for pug/dog-themed retirement meme or cartoon
- Space for image with caption
- Reference to client's Pugville Chronicles book

**Section 6: Call-to-Action**
- Prominent blue button: "Book Your Tax Savings Strategy Session >>"
- Linked to placeholder URL

**Footer Section**
- Feedback rating (thumbs up/down)
- Social media icon placeholders
- Compliance/disclaimer text
- Unsubscribe link placeholder

### 4. Email-Compatible HTML Export
- **Copy to Clipboard button** - One click copies the complete HTML
- Table-based layout for maximum email client compatibility
- Inline CSS (no external stylesheets)
- Tested structure for Outlook, Gmail, Apple Mail compatibility
- All links use placeholder URLs ({{CTA_LINK}}, {{UNSUBSCRIBE_LINK}}, etc.)

### 5. Technical Email Best Practices
- 600px max-width for desktop, fluid for mobile
- Web-safe fonts with fallbacks
- ALT text for all images
- VML fallbacks for Outlook where needed
- Proper `<table>` nesting for consistent rendering

---

## User Flow
1. Page loads with fully populated newsletter preview
2. User can edit any text content inline
3. User uploads/confirms logo (when ready)
4. User clicks "Copy HTML to Clipboard"
5. User pastes into their ESP (Mailchimp, Constant Contact, etc.)

---

## What You'll Get
A ready-to-use newsletter with professional content that you can review, tweak, and export directly to your email marketing platform.
